What Makes TikTok Emojis Different

TikTok uses a custom set of emojis that look slightly different from the standard emojis found on Android or iPhone. These custom emojis load directly from TikTok’s internal design library. They appear the same for every device, which makes communication consistent across the app.

Creators rely on these emojis for reactions, discussions, trends, and even storytelling. TikTok emojis are more expressive, sharper, and visually stronger in comments. This is why users prefer to include them in their posts instead of using device emojis.

How Secret TikTok Emoji Codes Work

TikTok has a hidden feature: typing certain words inside square brackets triggers special emojis. These are known as secret emoji codes.

Examples:

[smile] [cry] [angry] [surprised]

TikTok automatically converts these codes into its custom emojis. This feature works only inside the TikTok mobile app, not on browsers like Chrome or Safari. Many creators use these codes to add fun reactions. Since they are not available in the keyboard, users must type the exact code manually.
